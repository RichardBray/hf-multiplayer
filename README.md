# Online multiplayer haxeflixel game

## How this was built

https://www.youtube.com/watch?v=5XaGzRH5Y7I&t=1972s

## Install

This requires nodejs 16 or above

```sh
 npm i
```

## Run the client

```sh
 npm run dev
```

Run the command below in another tab:

```sh
 npm run server
```

## Run the server

Navigate to server/my-colyseus-server

```sh
 npm i
```

```sh
 npm start
```
